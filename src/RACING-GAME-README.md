# 🏎️ 极速赛车游戏

一个简单、好玩、速度快的网页版赛车游戏！

## 🎮 游戏特性

- **简单易上手**：使用方向键或 A/D 键控制赛车左右移动
- **速度快**：纯 Canvas 渲染，流畅运行
- **难度适中**：难度随分数增加而递增，最高 3 倍速
- **视觉效果**：精美的渐变背景、动态道路线条、彩色障碍车辆

## 📖 如何游玩

### 启动游戏
1. 直接在浏览器中打开 `racing-game.html` 文件
2. 点击 "开始游戏" 按钮

### 游戏操作
- **左移**：按 `←` 或 `A` 键
- **右移**：按 `→` 或 `D` 键
- **目标**：躲避迎面而来的障碍车辆，坚持越久分数越高

### 游戏规则
- 每成功躲避一辆车 +10 分
- 每得 100 分，游戏速度提升
- 撞到障碍车则游戏结束
- 最高速度：3 倍速

## 🎯 技术实现

### 核心技术
- **HTML5 Canvas**：用于游戏渲染
- **原生 JavaScript**：游戏逻辑实现
- **CSS3**：美化界面样式

### 游戏特点
1. **流畅性能**：使用 `requestAnimationFrame` 确保流畅的 60fps
2. **碰撞检测**：精确的矩形碰撞检测算法
3. **动态难度**：速度从 1x 逐渐增加到 3x
4. **响应式控制**：支持键盘多键同时操作

### 代码结构
```

    ├── 玩家车辆控制

    ├── 碰撞检测系统
    ├── 难度递增机制
    └── 游戏循环与渲染
```

## 💡 游戏提示

1. **保持节奏**：不要过度移动，观察障碍车的位置再做决定
2. **提前预判**：障碍车从顶部出现，有足够时间反应
3. **中央优势**：在中间车道更容易左右闪避
4. **速度适应**：随着速度提升，需要更快的反应速度
